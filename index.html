Projects

Some of the things I've done throughout the years..

  • NLP Classification for HCC Diagnosis

    Improved recall from 46% to 80% and achieved over 96% accuracy with a novel NLP model to extract and classify tumors from MRI Impressions.

  • MLFQ Scheduler for xv6

    Implemented a fully-functional MLFQ scheduler in xv6 with four priority queues. The scheduling method in each of the queues is round-robin, except for the bottom queue which uses FIFO.

  • Unix Shell

    Created a fully-operational, robust Unix shell from scratch in C. Uses various syscalls such as forking, execv, dup, and dup2. Enables users to input customizes paths and supports both piping and redirection.

  • MapReduce Architecture

    Implemented the backend to enable parallel computation from users.

  • Twitter Sentiment Analysis

    Achieved state-of-the-art accuracy by using a Rule-based approach to identify the sentiment of individual Tweets.

  • Filesystem Checker

    Wrote a filesystem checker for xv6 from scatch.
